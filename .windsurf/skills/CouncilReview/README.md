# CouncilReview Skill

## Purpose
Final oversight council review for comprehensive codebase validation and quality assurance.

## Functionality
- Performs 7-agent oversight council review
- Cross-validates findings from previous skills
- Ensures comprehensive accountability
- Provides final validation decision
- Enforces Penalty System if needed

## Usage
Final skill in audit workflow, invoked after ProjectHealthDashboard.
Requires user explicit approval through SkillControl workflow.

## Review Process
- Cross-checks YorkieFullValidation findings
- Validates ProjectHealthDashboard metrics
- Performs additional quality checks
- Ensures no issues were overlooked
- Applies Penalty System for violations

## Output
Final council review report with:
- Validation decision
- Penalty system triggers (if any)
- Comprehensive issue summary
- Final compliance determination
- Next-step recommendations

## Enforcement
- Triggers Penalty System for rule violations
- Ensures complete issue resolution
- Maintains Yorkie Constitution compliance
- Provides accountability for all findings

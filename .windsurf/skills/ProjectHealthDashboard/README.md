# ProjectHealthDashboard Skill

## Purpose
Generates comprehensive project health dashboard with metrics and visual indicators.

## Functionality
- Analyzes codebase health metrics
- Calculates project statistics
- Identifies code quality trends
- Generates visual health indicators
- Provides actionable insights

## Usage
Second skill in audit workflow, invoked after YorkieFullValidation.
Requires user explicit approval through SkillControl workflow.

## Health Metrics
- Code coverage percentage
- Test pass/fail rates
- Dependency health
- Code complexity metrics
- Security vulnerability count
- Performance indicators

## Output
Project health dashboard with:
- Overall health score
- Category-specific metrics
- Trend analysis
- Priority recommendations
- Visual indicators (when applicable)

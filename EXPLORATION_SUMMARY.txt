WHAT IS THIS PROJECT?
══════════════════════════════════════════════════════════════════════════

Jarvis v4 is a sophisticated multi-agent AI orchestration system built with 
TypeScript and Node.js. It manages 36+ specialized agents that handle different 
tasks (dialogue, music, finance, productivity, etc.) through a central 
orchestrator that intelligently routes requests.

Key Innovation: Self-healing infrastructure with diagnostic engines that can 
automatically detect and repair agent failures.

HOW TO BUILD, LINT, AND TEST
══════════════════════════════════════════════════════════════════════════

Prerequisites:
  - Node.js 20+ installed
  - npm installed

Build Process:

  # Full build
  npm install              # Install all dependencies
  npm run build            # Compile TypeScript to dist/
  npm run start            # Run compiled version

  # Development build with auto-reload
  npm run dev              # ts-node with hot reload
  npm run dev:watch        # nodemon for auto-recompile

Linting & Formatting:

  # Check for issues
  npm run lint             # Run ESLint
  npm run lint:fix         # Auto-fix ESLint issues
  npm run typecheck        # Type validation only
  npm run format:check     # Check Prettier formatting

  # Format code
  npm run format           # Apply Prettier formatting
  npm run lint:all         # Comprehensive lint check

  # Full quality check
  npm run quality          # Lint + analyze + quick tests

Code Analysis:

  npm run analyze              # Full analysis
  npm run analyze:deps         # Check dependencies
  npm run analyze:circular     # Find circular imports
  npm run analyze:complexity   # Check complexity
  npm run analyze:duplicate    # Find code duplication

Testing:

  # Quick verification
  npm run test:quick           # Smoke tests only
  npm run test:jest            # Jest unit tests

  # Comprehensive testing
  npm run test:all             # Full test suite
  npm run test:integration     # Integration tests
  npm run test:performance     # Performance benchmarks
  npm run test:stress          # Stress testing
  npm run test:security        # Security audit
  npm run test:data-integrity  # Data integrity checks

  # Continuous monitoring
  npm run monitor:health       # Long-running health check

Production Deployment:

  npm run build                # Compile
  npm run pm2:start            # Start with PM2
  npm run pm2:logs             # View logs
  npm run pm2:status           # Check status

Quick Start for Development:

  1. npm install               # Install dependencies
  2. npm run typecheck         # Verify setup (ignore some errors)
  3. npm run dev               # Start development server
  4. npm run test:quick        # Verify tests work
  5. npm run lint              # Check code quality

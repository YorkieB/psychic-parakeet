## How to Use

### Option 1: Run Directly (No Installation)
1. Navigate to: `jarvis-desktop\release\win-unpacked\`
2. Double-click `Jarvis AI Assistant.exe`
3. The app will launch immediately

### Option 2: Create Desktop Shortcut
1. Right-click `Jarvis AI Assistant.exe`
2. Select "Create shortcut"
3. Move the shortcut to your Desktop
4. Double-click the shortcut to launch

### Option 3: Copy to Anywhere
You can copy the entire `win-unpacked` folder to:
- `C:\Program Files\Jarvis\` (for system-wide access)
- `C:\Users\YourName\AppData\Local\Jarvis\` (for user-only access)
- Any other location you prefer

**Important:** Keep all files in the `win-unpacked` folder together - don't move just the .exe file!

## What's Inside

The `win-unpacked` folder contains:
- `Jarvis AI Assistant.exe` - Main application
- `resources/` - App resources and assets
- `locales/` - Language files
- Other Electron runtime files

All these files must stay together for the app to work.

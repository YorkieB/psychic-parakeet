## System Overview

### How NRS Works

```
[NRS-###] Format:
┌─────────────┬──────────────┬──────────────┐
│   [NRS-     │      ###     │      ]       │
├─────────────┼──────────────┼──────────────┤
│   Prefix    │   3 Digits   │   Suffix     │
└─────────────┴──────────────┴──────────────┘

Example: [NRS-205]
         ↓
         NRS-2xx = YAML Parsing/Loading
                2xx = Parsing category
                 05 = load() function
```
